To access and modify resources, you can use any HTTP method
GET POST PUT PATCH DELETE OPTIONS

Example

  • GET http://localhost:3000/api/categories Get all categories
  • GET http://localhost:3000/api/categories/1 Get a category by Id 1
  • POST http://localhost:3000/api/categories Adds a new category
  • PUT http://localhost:3000/api/categories Edit a category
  • DELETE http://localhost:3000/api/categories/1 Delete a category with Id 1
